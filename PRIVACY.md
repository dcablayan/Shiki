# Shiki Privacy Policy

Last updated: June 19, 2026

Shiki is a browser extension from Lapetus Software that changes the local interface of supported AI chat websites. It has no Shiki account system, backend, analytics, ads, or telemetry.

## Data Shiki handles

Shiki handles data from supported AI chat pages so it can render and control the document-style workspace in your browser. This can include conversation titles, message text, model labels, image URLs or image data already present on the page, and page controls needed to switch chats or submit prompts.

Shiki also handles data you provide directly to the extension, including prompts typed into Shiki's composer, images attached through Shiki before sending, a custom profile picture if you set one, and local organization settings such as pinned chats, aliases, hidden chats, display mode, and formatting preferences.

## How Shiki uses data

Shiki uses this data only to provide its single purpose: turning supported AI chat pages into a focused document workspace in your browser.

For example, Shiki reads the current page to render your conversation, sends prompts or image attachments to the AI site's own composer when you ask it to, switches conversations using the site's existing controls, and stores your local preferences so the workspace behaves consistently across tabs.

## Storage

Shiki stores settings and local-only organization data in your browser's local extension storage on your device. If you set a custom profile picture, that image is also stored locally in browser extension storage.

Uninstalling the extension removes that browser's extension storage. You can also reset the custom profile picture from the extension popup.

## Network and sharing

Shiki does not send data to Lapetus Software-controlled servers and does not sell or transfer your data for advertising, analytics, profiling, or data brokerage.

When you use Shiki to send a prompt or attach an image, that content is submitted to the AI service you are already using through that service's own webpage. Shiki may also display images from the AI page using image sources already present on that page, including HTTPS, blob, or data image sources. Those requests go to the relevant AI service or image host, not to Shiki's developer.

Shiki does not allow Lapetus Software or any Shiki operator to read your conversations or local extension data.

## Third-party services

Shiki works only on the supported AI chat domains declared by the extension: `chatgpt.com`, `chat.openai.com`, `claude.ai`, and `gemini.google.com`. Those services are operated by their respective providers and are governed by their own terms and privacy policies. Shiki does not control how those services process data you enter into their websites.

## Limited Use statement

For Chrome distribution, Shiki's use of information received from Chrome extension permissions adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements. Shiki uses data only to provide or improve its single purpose, does not transfer user data except as needed to perform user-requested actions on the supported AI site, does not use user data for advertising, and does not allow humans to read user data except if required by law or with explicit user consent for support.
